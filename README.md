# NVM Burst Buffer

## Getting Started

### Install
```
$ git clone --recursive https://github.com/guddnjs91/autobahn_app.git
```
or
```
$ git clone https://github.com/guddnjs91/autobahn_app.git
$ git submodule init
$ git submodule update
```
#### libcuckoo
refer to https://github.com/efficient/libcuckoo

### Compile
```
$ make
```

### Run
```
$ bin/test
```

### Clean
```
$ make clean
$ bin/shm-remove
```
